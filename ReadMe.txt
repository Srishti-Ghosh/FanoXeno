vpn
    The folder contains a set of fortran programs for calculating 
    the scattering phaseshifts, angle-differential elestic scattering cross sections,
    and associating w- and q- paramewters.
    The folder is a copy of the directory that works under a linux or its equivalent environment
    that have a gfortran availability.
    The excecutable program: difelsccscan be generated by giving a command
       make -f Makefile

    To compile the code under any different environment, pls refer to the description
    in the file: Makefile, where one can find the list of fortran program files that should be compiled.

    Pls. note here that the code-set in this folder is tuned for the calculation of the 
    specific system: H+Kr. And therefore one must change the physical parameters 
    as to fit to the systems for which one wants to calculate.

    The set of original library files are stored in a folder named orig.pkgs.
    Here also one can find the papers that describe the code usage.

H-Kr
    Details of the actual calculations for  H+Kr collision system are given.
    After replacing the physical parameters of H+Kr system by the ones
    of the other systems, one can use the files for production of the data or figures.

    This entry contains the following folders:

    Calculation
        DCS
           Differential cross sections can be calculated by giving a command
              difelsccs <input.DCS 
           where difelsccs is the excecutable program that is compiled in the vpn folder.
        WPRM
           w-parameter, q-parameter can be calculated by entering a command
              difelsccs < input.WPRM
           where difelsccs is the excecutable program that is compiled in the vpn folder.

    AngleDCS
          angledcs.plt: gnuplot command script file for drawing the angle differetial cross sections
          angledcs.animation.plt: gnuplot command script file for drawing the animation of the angle differetial cross sections

    Phaseshift, Potential, WprmGauss, WprmSteric, WprmWithProfile
         All the *.plt files are gnuplot command script files to generate relevant figure files.
